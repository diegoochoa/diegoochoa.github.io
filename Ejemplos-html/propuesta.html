Propuesta de proyecto de Tecnologias Web

Para el proyecto haremos una agenda en linea que permita a los usuarios buscar asesorias por materias, y que puedas agendar una cita con el tutor encargado de dicha asesoria.

Se podra registrar como usuario o como tutor, por lo cual seran distintas pantallas para cada tipo de registro, tanto los usuarios como los tutores podran agregar notas a la asesoria con el fin de llevar un control sobre ellas.

Se espera poder utilizar este proyecto para fines practicos en el area, asi promover la ayuda entre alumnos y profesores.
