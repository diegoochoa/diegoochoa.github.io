Introduccion a HTML

Hypertext Markup Language

Todas las paginas web son documentos de texto
HTML es un lenguaje de texto mejorado.

Todos los documentos estan compuestos por elementos
Todos los elementos tienen la siguiente estructura:
<etiqueta> contenido </etiqueta>

Existen algunos elementos vacios: <etiqueta> Por ejemplo: <br>

Los elementos pueden incluir atributos que contienen informacion adicional al elemento.
<etiqueta nombre="valor"> contenido </etiqueta>
