Introduccion a HTML

Hypertext Markup Language

Todas las paginas web son documentos de texto
HTML es un lenguaje de texto mejorado.

Todos los documentos estan compuestos por elementos
Todos los elementos tienen la siguiente estructura:
<etiqueta> contenido </etiqueta>

Existen algunos elementos vacios: <etiqueta> Por ejemplo: <br>

Los elementos pueden incluir atributos que contienen informacion adicional al elemento.
<etiqueta nombre="valor"> contenido </etiqueta>

CSS

Es un lenguaje para describir la presentacion de las paginas web, incluyendo color, diseño y fientes. (W3C)

Permite adaptar la presentacion a diferentes tipos de dispositivos, tales como pantallas grandes, pantallas pequeñas e impresoras.

Es independiente de HTML, puede ser usado con cualquier lenguaje de marcado basado en XML

La separacion de HTML y CSS hace mas facil manejar sitios, compartir hojas de estilo entre paginas y ajustarlas a diferentes entornos.

Versiones Borrador Liberado Soportado
-CSS1 1996 1999 2001
-CSS2 1998 2007
-CC3 2001

CSS esta basado en Reglas

-Sintaxis de una regla:
selector{declaraciones;}

Sintaxis de declaraciones:
propiedad:valor;
Ejemplo: h1{color:blue;}

Tipos de estilos:

  1. Por default (aplicado por el navegador)
  2. Definidos por el usuario ("user styles")
  3. Hojas de estilo externos
  4. Estilo interno (etiqueta <style> de HTML)
  5. Estilo de linea (atributo style de HTML)

Tipos de selectores:

  1. Selector universal *
    2. Aplica un estilo a todos los elementos del documento.
    Ejemplo:
    *{font-family : Arial;}

  2. Selector de tipo (etiqueta)
    3. Aplica un estilo a todos los elementos de la etiqueta.
    Ejemplo:
    h1{......;}

  3. Selector de base
    4. Aplica un estilo a los elementos que tengan el atributo class especificado por un valor.
    Ejemplo:
    .nota{color: green;}

  4. Selector de ID
    5. Aplica el estilo a los elementos que tienen el atributo ID con un valor dado.
    Ejemplo:
    #ID {background-color: cyan;}
